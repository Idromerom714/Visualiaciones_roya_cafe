# Visualizaciones_roya_cafe
Dashboard web para visualizaciones y analisis de la roya en los cultivos de café.

Como ejecutarlo?

1. Clona el repositorio (para abrirlo en Visual Studio code, por ejemplo)
2. En la terminal, ejecuta el comando (streamlit run app.py), si esto no funciona, prueba primero (pip install requirements.txt)
3. Lo anterior te generará 3 enlaces, selecciona la opción llamada localhost
4. Se abrirá en tu navegador el dashboard (en modo local)
